## Como jogar?
1. Instalação do NASM e do QEMU:
  ```
  sudo apt update
  ```
  ```
  sudo apt install nasm
  ```
  ```
  sudo apt install qemu
  ```
2. Clonar este repositório:
  ```
  git clone 'https://github.com/acrucha/Enigmas-Enigmaticos'
  ```
3. Compilar e rodar o jogo:
  ```
  make all
  ```
